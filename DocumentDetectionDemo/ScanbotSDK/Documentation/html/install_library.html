Installing Scanbot SDK as a static linked library
Step 1: Drag the Scanbot SDK folder to your project
Step 2: Select 'Copy items if needed' and 'Create groups' and hot 'Finish'
Step 3: Verify settings
Step 4: Add CoreMotion framework to your project
Step 5: Add AssetsLibrary framework to your project
Step 6: Add libc++.dylib to your project
Step 7: Set '-ObjC' linker flag
Step 8: Make sure that ScanbotDSK.bundle is copied into the apps resources folder
